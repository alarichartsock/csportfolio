## Quick Start

### Dev

1. Activate Python Virtual Environment 
2. pip install -r requirements.txt
3. pip install -r requirements.dev
4. python manage.py runserver

### Prod

1. Activate Python Virtual Environ ment
2. pip install -r requirements.txt
3. make deploy
